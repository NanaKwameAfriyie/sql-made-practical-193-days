# SQL Made Practical: 193 Days of Real Business Problems

A public SQL learning and teaching journey by **Nana Kwame Owusu-Afriyie**.

From 22 June to 31 December 2026, I am teaching practical SQL through daily lessons, real business questions, debugging exercises, case studies and portfolio projects.

The purpose of this repository is to document the complete journey and provide accessible resources for people learning how SQL is used in professional data work.

## The Teaching Framework

Every lesson follows the same problem-solving process:

**Question → Grain → Query → Validation → Business Decision**

This means that the objective is not simply to write SQL that runs.

Each lesson will examine:

1. The business question being answered
2. What one row represents in the relevant tables
3. The SQL logic required
4. How the result should be validated
5. What decision the analysis can support

## What the Journey Will Cover

The 193-day journey will progress through:

* SQL foundations
* Filtering, aggregation and conditional logic
* Joins and relational problem-solving
* Subqueries and common table expressions
* Window functions
* Advanced analytical SQL
* Customer, revenue, retention and operational analysis
* Data modelling
* Analytics engineering
* dbt, Git, testing and documentation
* Query optimisation
* Production SQL
* SQL interview challenges
* Portfolio projects
* AI-assisted SQL and data-quality validation
